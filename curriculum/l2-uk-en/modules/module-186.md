## Практика

### Заповніть пропуски

1. Лікар ___ (оглянути) пацієнта.
   → **оглянув**

2. Йому ___ (призначити) антибіотики.
   → **призначили**

3. Після операції він проходить ___.
   → **реабілітацію**

4. Ліки мають ___ дію.
   → **побічну**

5. Пацієнта ___ (госпіталізувати) з пневмонією.
   → **госпіталізували**

### Визначте термін

1. Лікар, який оперує → **хірург**
2. Дослідження за допомогою рентгенівських променів → **рентген**
3. Лікування за допомогою ліків → **медикаментозне**
4. Письмовий дозвіл на отримання ліків → **рецепт**
5. Негативна реакція на ліки → **побічна дія / алергія**

### Перекладіть

1. The doctor prescribed antibiotics.
   → **Лікар призначив антибіотики.**

2. The patient was hospitalized with pneumonia.
   → **Пацієнта госпіталізували з пневмонією.**

3. Take this medicine three times a day.
   → **Приймайте ці ліки тричі на день.**

# Вправи

## quiz: Медична термінологія

> Виберіть правильну відповідь.

1. «УЗД» — це:
   - [x] ультразвукове дослідження
   - [ ] рентген
   > Ultrasound.

2. «Призначити ліки» означає:
   - [x] to prescribe medicine
   - [ ] to take medicine
   > Лікар призначає.

3. «Побічна дія» — це:
   - [x] side effect
   - [ ] main effect
   > Негативний ефект.

4. «Госпіталізувати» означає:
   - [x] покласти до лікарні
   - [ ] виписати з лікарні
   > To hospitalize.

5. «Терапевт» — це:
   - [x] лікар загальної практики
   - [ ] хірург
   > General practitioner.

6. **[Інтеграція: пасив]** Медичний звіт: «Пацієнту ___ діагноз.»
   - [x] поставлено
   - [ ] поставили
   - [ ] ставиться
   > Офіційний медичний стиль → -но/-то.

7. **[Інтеграція: регістр]** Розмова: «Мені ___ антибіотики.»
   - [ ] призначено
   - [x] призначили
   - [ ] призначається
   > Розмовний стиль → 3-я особа мн.
