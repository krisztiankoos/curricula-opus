# Зміст уроку

## Розминка

### Як перекласти?

Англійська: **I was told** that...
Українська: ???

| Варіант | Стиль |
|---------|-------|
| Мені **сказали**, що... | Розмовний ✓ |
| Мені було сказано, що... | Офіційний |

В українській розмовній мові **3-я особа множини** — найприродніший спосіб виразити пасив!

## Презентація

### Структура

**Англійський пасив → Українська 3-я ос. мн.**

| Англійська | Українська |
|------------|------------|
| I was told... | Мені **сказали**... |
| He was asked... | Його **запитали**... |
| We were shown... | Нам **показали**... |
| They were invited... | Їх **запросили**... |

**Структура:**
```
Додаток (Давальний/Знахідний) + Дієслово 3-ї ос. мн.
```

### Відмінки додатка

**Залежить від дієслова:**

| Дієслово | Відмінок | Приклад |
|----------|----------|---------|
| сказати (кому?) | Давальний | **Мені** сказали. |
| запитати (кого?) | Знахідний | **Мене** запитали. |
| показати (кому?) | Давальний | **Нам** показали. |
| запросити (кого?) | Знахідний | **Їх** запросили. |
| допомогти (кому?) | Давальний | **Йому** допомогли. |
| попросити (кого?) | Знахідний | **Її** попросили. |

### Часові форми

| Час | Приклад |
|-----|---------|
| Минулий | Мені **сказали**. |
| Теперішній | Мені **кажуть**. |
| Майбутній | Мені **скажуть**. |

**Парадигма:**

| Особа | Минулий | Теперішній | Майбутній |
|-------|---------|------------|-----------|
| 3 мн. | сказали | кажуть | скажуть |
| 3 мн. | запитали | запитують | запитають |
| 3 мн. | показали | показують | покажуть |

### Типові дієслова

| Дієслово | Пасивний зміст | Приклад |
|----------|----------------|---------|
| сказати | to be told | Мені сказали новину. |
| запитати | to be asked | Мене запитали про це. |
| попросити | to be asked (request) | Мене попросили зачекати. |
| запросити | to be invited | Нас запросили на свято. |
| показати | to be shown | Їм показали дорогу. |
| дати | to be given | Мені дали книгу. |
| пояснити | to be explained | Нам пояснили правила. |
| навчити | to be taught | Мене навчили плавати. |
| попередити | to be warned | Їх попередили про небезпеку. |
| повідомити | to be informed | Нас повідомили про зміни. |

### Порівняння стилів

| Розмовний (3-я ос.) | Офіційний (-но/-то) |
|---------------------|---------------------|
| Мені сказали. | Мені було сказано. |
| Його запитали. | Його було запитано. |
| Нас запросили. | Нас було запрошено. |
| Їм показали. | Їм було показано. |

> 💡 **Природність**
>
> 3-я особа множини — найприродніша форма в розмовній українській.
> Форми «мені було сказано» звучать надто офіційно для щоденного спілкування.

### Діяч (необов'язковий)

Діяч зазвичай **не вказується**, бо він невідомий або неважливий:

| З діячем | Без діяча (типово) |
|----------|-------------------|
| Мені сказали **друзі**. | Мені сказали. |
| Його запитали **поліцейські**. | Його запитали. |

### Переклад з англійської

| Англійська | Українська |
|------------|------------|
| I was born in Kyiv. | Я народився в Києві. (зворотне) |
| I was told the news. | Мені **сказали** новину. |
| She was given a book. | Їй **дали** книгу. |
| We were shown the way. | Нам **показали** дорогу. |
| They were asked to wait. | Їх **попросили** зачекати. |
| He was taught to swim. | Його **навчили** плавати. |

> ⚠️ **Виняток: «народитися»**
>
> «I was born» перекладається як «Я **народився**» (зворотне дієслово), НЕ «Мене народили».

## Практика

### Перекладіть

1. I was told about the meeting.
   → Мені **сказали** про зустріч.

2. She was asked many questions.
   → Її **запитали** багато питань.

3. We were invited to the party.
   → Нас **запросили** на вечірку.

4. He was shown the new office.
   → Йому **показали** новий офіс.

5. They were warned about the danger.
   → Їх **попередили** про небезпеку.

### Виберіть відмінок

1. ___ (я) сказали про це.
   → **Мені** (Давальний)

2. ___ (він) запитали про роботу.
   → **Його** (Знахідний)

3. ___ (ми) показали місто.
   → **Нам** (Давальний)

4. ___ (вона) попросили допомогти.
   → **Її** (Знахідний)

5. ___ (вони) дали квитки.
   → **Їм** (Давальний)

### Трансформуйте стиль

Перетворіть офіційний стиль на розмовний:

1. Мені було повідомлено про зміни.
   → Мені **повідомили** про зміни.

2. Її було запрошено на конференцію.
   → Її **запросили** на конференцію.

3. Нам було пояснено правила.
   → Нам **пояснили** правила.

## Продукція

### Розкажіть історію

Розкажіть, що вам сказали/показали/дали, використовуючи 3-ю ос. мн.:

**Приклад:**
«Учора мені **зателефонували** з роботи. Мені **сказали**, що потрібно прийти раніше. Коли я прийшов, мені **показали** новий проєкт. Мені **пояснили** завдання і **дали** документи.»

# Підсумок

## Невизначено-особовий пасив

| Аспект | Правило |
|--------|---------|
| Форма | Дієслово 3-ї особи множини |
| Стиль | Розмовний |
| Діяч | Зазвичай не вказується |
| Відмінок | Залежить від дієслова |

## Типові переклади

| Англійська | Українська |
|------------|------------|
| I was told | Мені сказали |
| I was asked | Мене запитали |
| I was shown | Мені показали |
| I was given | Мені дали |

> **Головне**
>
> - 3-я особа мн. — найприродніший пасив у розмовній мові
> - Відмінок додатка залежить від керування дієслова
> - Діяч зазвичай не вказується
> - «I was born» → «Я народився» (виняток!)
