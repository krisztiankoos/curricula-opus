# Зміст уроку

## Розминка

### 1569 рік

Що означала Люблінська унія для України?

Прочитайте:
«Люблінська унія 1569 року об'єднала Польщу та Литву в єдину державу — Річ Посполиту. Українські землі перейшли безпосередньо під польську корону.»

## Презентація

### Люблінська унія (1569)

**Створення Речі Посполитої:**

| Факт | Деталі |
|------|--------|
| Дата | 1 липня 1569 |
| Місце | Люблін |
| Назва | Річ Посполита (Res Publica) |
| Структура | Корона (Польща) + Литва |

**Що змінилося для України:**

| До унії | Після унії |
|---------|------------|
| Під Литвою | Під Польщею |
| Руська мова | Польська мова |
| Автономія | Воєводства |
| Православ'я | Тиск католицизму |

### Українські воєводства

| Воєводство | Столиця |
|------------|---------|
| Руське | Львів |
| Белзьке | Белз |
| Волинське | Луцьк |
| Подільське | Кам'янець |
| Брацлавське | Брацлав |
| Київське | Київ |

### Панщина та кріпацтво

**Закріпачення селян:**

| Рік | Закон |
|-----|-------|
| 1496 | Заборона селянам залишати село |
| 1505 | Обмеження прав |
| 1573 | Збільшення панщини |
| XVI-XVII ст. | Повне закріпачення |

**Що таке панщина?**
- Примусова праця на пана
- Спочатку 1-2 дні на тиждень
- Зросла до 5-6 днів на тиждень
- Селяни — власність пана

**Літописне свідчення:**
«Селяни працювали на пана шість днів на тиждень. Лише неділя була їхня. Пан міг продати, обміняти або покарати селянина без суду.»

### Релігійний конфлікт

**Брестська унія (1596):**

| Факт | Деталі |
|------|--------|
| Рік | 1596 |
| Суть | Частина православної церкви приєдналася до Риму |
| Назва | Греко-католицька (уніатська) церква |
| Причина | Тиск Польщі, бажання рівних прав |

**Наслідки унії:**
| Група | Позиція |
|-------|---------|
| Уніати | Підтримали унію |
| Православні | Відкинули унію |
| Козаки | Захищали православ'я |

**Конфлікт:**
«Унія розколола українське суспільство. Православні братства боролися за свої права. Козаки стали захисниками православної віри.»

### Культурний спротив

**Братства:**
- Освітні та релігійні організації
- Школи, друкарні
- Захист православ'я

| Братство | Місто | Діяльність |
|----------|-------|------------|
| Львівське | Львів | Школа (1586), друкарня |
| Острозьке | Острог | Академія (1576), Біблія |
| Київське | Київ | Школа, Лавра |

**Острозька Біблія (1581):**
- Перша повна Біблія церковнослов'янською
- Надрукована Іваном Федоровим
- Культурне досягнення

**Києво-Могилянська академія (1632):**
- Заснована Петром Могилою
- Перший вищий навчальний заклад
- Латина, грецька, церковнослов'янська
- Центр української освіти

### Полонізація

**Що таке полонізація?**
- Поширення польської мови та культури
- Перехід шляхти в католицизм
- Втрата національної ідентичності

**Наслідки:**
| Явище | Приклад |
|-------|---------|
| Мова | Руська еліта говорила польською |
| Релігія | Перехід у католицизм |
| Прізвища | Полонізація прізвищ |
| Культура | Польські звичаї |

**Приклад:**
«Рід Вишневецьких був руським православним родом. За кілька поколінь Вишневецькі стали поляками-католиками. Єремія Вишневецький воював проти козаків.»

### Ключові терміни

| Термін | Переклад |
|--------|----------|
| Річ Посполита | Commonwealth |
| панщина | serfdom, corvée |
| кріпак | serf |
| шляхта | nobility |
| магнат | magnate |
| унія | union |
| братство | brotherhood |
| полонізація | Polonization |

### Чому це важливо?

**Деколонізаційний контекст:**

1. **Колоніальний досвід:** Польське панування — форма колоніалізму.

2. **Культурний спротив:** Братства та козаки зберегли ідентичність.

3. **Причина повстань:** Панщина та релігійний тиск призвели до козацьких повстань.

## Практика

### Заповніть пропуски

1. Люблінська унія підписана у ___ році.
   → **1569**

2. Брестська унія створила ___ церкву.
   → **греко-католицьку / уніатську**

3. Острозьку Біблію надруковано у ___ році.
   → **1581**

4. Києво-Могилянську академію заснував ___.
   → **Петро Могила**

5. Примусова праця на пана називається ___.
   → **панщина**

### Визначте правильне твердження

1. ✅ Після 1569 року українські землі перейшли під Польщу.
2. ❌ Брестська унія об'єднала всіх православних.
3. ✅ Братства захищали православ'я та освіту.
4. ✅ Полонізація означала перехід на польську мову.
5. ❌ Панщина була добровільною.

### Перекладіть

1. Serfdom became widespread in the 16th century.
   → **Кріпацтво поширилося у XVI столітті.**

2. The brotherhoods defended Orthodox culture.
   → **Братства захищали православну культуру.**

3. The Ostroh Bible was the first complete Slavonic Bible.
   → **Острозька Біблія була першою повною слов'янською Біблією.**
