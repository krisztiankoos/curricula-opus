# Warm-up

Learn to express emotions and feelings in Ukrainian! This module teaches how to say "I'm happy that...", "It's a pity that...", and other emotion expressions.

---

# Presentation

## Emotion + що Structures

| Expression | Meaning |
|------------|---------|
| Я радий/рада, що... | I'm happy that... |
| Мені сумно, що... | I'm sad that... |
| Шкода, що... | It's a pity that... |
| Добре, що... | It's good that... |
| Погано, що... | It's bad that... |
| Приємно, що... | It's nice that... |

**Examples:**
- **Я радий, що** ти прийшов. (I'm happy that you came.)
- **Шкода, що** він не може. (It's a pity that he can't.)
- **Добре, що** ти тут. (It's good that you're here.)
