## Summary of Gaps

### Critical Gaps (Must Add)
1. **Euphony rules (у/в, і/й, з/із/зі)** - Currently in A2 (37-38), should be explicit in A1
2. **Word formation (словотвір)** - Needs dedicated B2 modules:
   - Person suffixes (-ач, -ець, -ник)
   - Activity nouns (-ння, -ення)
   - Place names (-ня: читальня, спальня)
   - Adjectives from nouns (-ський)
3. **Complex numeral declension** - Needs B2 expansion beyond B1 basics

### Minor Gaps (Should Add)
4. **Stress patterns (наголос)** - Should be explicit module in A1
5. **Intonation patterns** - Missing explicit teaching
6. **Antonyms/Omonyms** - Need explicit B2 treatment
7. **Metaphors/Epithets** - Need B2 stylistics modules
8. **Advanced punctuation** - Missing in C1
9. **C2 level** - Not currently planned (beyond scope?)

### C1 Module Numbering Issue
The C1 curriculum plan shows modules 233-352, but based on our structure it should be **311-400**. This needs to be corrected.

---

## Recommendations

1. **Add A1 module** for euphony/stress/intonation (or integrate into module 2)
2. **Add B2 word formation block** (4-5 modules on словотвір)
3. **Expand B2 numerals** beyond B1 basics
4. **Add B2 stylistic devices** modules (metaphors, epithets, etc.)
5. **Fix C1 module numbers** (233-352 → 311-400)
6. **Consider C2 scope** - may be beyond initial release
