### Приклад тексту

**Судова новина:**
«Апеляційний суд залишив без змін вирок місцевого суду у справі колишнього посадовця. Захист планує подати касаційну скаргу до Верховного Суду. За словами адвоката, є підстави для перегляду справи. Прокуратура наполягає на законності вироку.»

## Практика

### Заповніть пропуски

1. Суд ___ (винести) вирок у справі.
   → **виніс**

2. Адвокат планує ___ (оскаржити) рішення.
   → **оскаржити**

3. Підсудного ___ (визнати) винним.
   → **визнали**

4. Прокурор ___ (пред'явити) обвинувачення.
   → **пред'явив**

5. Потерпілий ___ (подати) позов.
   → **подав**

### Визначте термін

1. Людина, проти якої порушено справу → **підсудний / обвинувачений**
2. Юрист, що захищає в суді → **адвокат / захисник**
3. Рішення суду про покарання → **вирок**
4. Оскарження рішення суду → **апеляція**
5. Невизнання винним → **виправдання**

### Перекладіть

1. The court found him guilty.
   → **Суд визнав його винним.**

2. The lawyer filed an appeal.
   → **Адвокат подав апеляцію.**

3. He was sentenced to 3 years in prison.
   → **Його засудили до 3 років позбавлення волі.**

# Вправи

## quiz: Право та правосуддя

> Виберіть правильну відповідь.

1. «Підсудний» — це:
   - [x] обвинувачений у кримінальній справі
   - [ ] суддя
   > Підсудний = defendant.

2. «Винести вирок» означає:
   - [x] оголосити рішення суду
   - [ ] подати позов
   > Вирок = verdict, sentence.

3. «Апеляція» — це:
   - [x] оскарження рішення суду
   - [ ] перше засідання
   > Апеляція = appeal.

4. Хто захищає підсудного в суді?
   - [x] адвокат
   - [ ] прокурор
   > Адвокат = defense lawyer.

5. «Презумпція невинуватості» означає:
   - [x] людина невинна, поки не доведено інше
   - [ ] людина завжди винна
   > Presumption of innocence.

6. **[Інтеграція: пасив + регістр]** Офіційний протокол: «Вирок ___»
   - [x] винесено
   - [ ] винесли
   - [ ] виноситься
   > Офіційний документ → -но/-то форма.

7. **[Інтеграція: пасив + регістр]** Наукова стаття: «Проблема корупції ___ у багатьох дослідженнях.»
   - [ ] досліджено
   - [ ] досліджували
   - [x] досліджується
   > Науковий стиль, процес → -ся форма.

# Підсумок

## Судова система

| Рівень | Суд |
|--------|-----|
| Найвищий | Верховний Суд |
| Апеляційний | Апеляційні суди |
| Першої інстанції | Місцеві суди |

## Ключові терміни

| Термін | Значення |
|--------|----------|
| вирок | рішення суду |
| апеляція | оскарження |
| виправдання | невизнання винним |
| засудити | признати винним і покарати |

> **Головне**
>
> - Суддя виносить вирок
> - Адвокат захищає, прокурор обвинувачує
> - Апеляція — право оскаржити рішення
> - Презумпція невинуватості — основа права
