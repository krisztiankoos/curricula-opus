# Зміст уроку

## Розминка

### Ти чи Ви?

Українська має **дві системи звертання:**

| Форма | Вживання |
|-------|----------|
| **Ти** | Друзі, родина, діти, ровесники |
| **Ви** | Незнайомці, старші, офіційні ситуації |

> ⚠️ **Важливо**
>
> «Ви» до однієї особи пишеться з **великої літери** в листах та офіційних текстах.

## Презентація

### Система «ти/Ви»

**Коли «ти»:**
- Друзі та близькі знайомі
- Родина (батьки, діти, брати, сестри)
- Діти (до ~14 років)
- Ровесники в неформальних ситуаціях
- Тварини

**Коли «Ви»:**
- Незнайомці
- Старші за віком
- Начальство, викладачі
- Офіційні ситуації
- Публічне звертання
- Перше знайомство

### Граматичні відмінності

| Елемент | Неформальний (ти) | Формальний (Ви) |
|---------|-------------------|-----------------|
| Підмет | ти | Ви |
| Дієслово | 2 ос. одн. | 2 ос. мн. |
| Присвійний | твій/твоя | Ваш/Ваша |
| Імператив | дай, скажи | дайте, скажіть |

**Приклади:**

| Неформальний | Формальний |
|--------------|------------|
| Як **ти** **думаєш**? | Як **Ви** **думаєте**? |
| Це **твій** телефон? | Це **Ваш** телефон? |
| **Дай** мені книгу. | **Дайте** мені книгу, будь ласка. |
| **Підожди** хвилинку. | **Зачекайте**, будь ласка. |

### Лексичні маркери

**Неформальний регістр:**

| Категорія | Приклади |
|-----------|----------|
| Звертання | Привіт! Слухай! Гей! |
| Прощання | Бувай! Па! До зустрічі! |
| Згода | Добре! Ок! Звісно! |
| Заперечення | Ні, не можу. Та ні! |
| Подяка | Дякую! Дякс! |
| Вибачення | Вибач! Сорі! |

**Формальний регістр:**

| Категорія | Приклади |
|-----------|----------|
| Звертання | Добрий день! Шановний пане/пані! |
| Прощання | До побачення! На все добре! |
| Згода | Так, звичайно. Погоджуюсь. |
| Заперечення | На жаль, не маю змоги. |
| Подяка | Дякую Вам! Вдячний(-а)! |
| Вибачення | Перепрошую! Вибачте! |

### Звертання по імені

**Неформальне:**
- Петре! Оксано! (кличний відмінок)
- Ім'я без по батькові

**Формальне:**
- Пане Петре! Пані Оксано!
- Ім'я + по батькові: Петре Івановичу! Оксано Михайлівно!

### Письмові формули

**Початок листа:**

| Неформальний | Формальний |
|--------------|------------|
| Привіт, Петре! | Шановний пане Петренко! |
| Вітаю! | Вельмишановний пане директоре! |

**Кінець листа:**

| Неформальний | Формальний |
|--------------|------------|
| Бувай! | З повагою, |
| Обіймаю! | З найкращими побажаннями, |
| Твій друг, | Щиро Ваш(-а), |

### Прохання

| Неформальне | Формальне |
|-------------|-----------|
| Дай мені... | Чи не могли б Ви дати мені... |
| Допоможи! | Прошу Вас допомогти. |
| Зроби це! | Чи могли б Ви це зробити? |
| Скажи... | Будь ласка, скажіть... |

### Перехід на «ти»

Як перейти від «Ви» до «ти»:

**Формула:** «Можемо перейти на ти?» / «Давай на ти!»

**Правила:**
1. Старший пропонує молодшому
2. Жінка пропонує чоловікові (традиційно)
3. Начальник — підлеглому
4. Обидві сторони мають погодитися

**Приклад діалогу:**
— Може, перейдемо на «ти»?
— Залюбки! Тоді я — Петро.
— А я — Оксана. Дуже приємно!

## Практика

### Трансформуйте

**Неформальний → Формальний:**

1. Привіт! Як справи?
   → **Добрий день! Як Ваші справи?**

2. Слухай, можеш допомогти?
   → **Вибачте, чи не могли б Ви мені допомогти?**

3. Дякую! Бувай!
   → **Дякую Вам! До побачення!**

**Формальний → Неформальний:**

1. Шановний пане Іване!
   → **Привіт, Іване!**

2. Чи не могли б Ви передати документи?
   → **Передай документи, будь ласка.**

3. Вдячна за Вашу допомогу.
   → **Дякую за допомогу!**

### Виберіть регістр

**Ситуація 1:** Розмова з другом дитинства
→ **Неформальний**: «Привіт! Як ти? Давно не бачилися!»

**Ситуація 2:** Лист до викладача
→ **Формальний**: «Шановний пане професоре! Звертаюся до Вас з проханням...»

**Ситуація 3:** Розмова з бабусею
→ **Неформальний**: «Бабусю, як твоє здоров'я?»

**Ситуація 4:** Телефонний дзвінок у банк
→ **Формальний**: «Добрий день! Чи могли б Ви мені допомогти з...»

## Продукція

### Напишіть повідомлення

Напишіть одне повідомлення у двох версіях:

**Тема:** Запрошення на день народження

**Неформальне (другу):**
«Привіт! У суботу святкую день народження. Приходь о 18:00! Будемо весело!»

**Формальне (колезі):**
«Добрий день! Запрошую Вас на святкування мого дня народження у суботу о 18:00. Буду радий(-а) Вас бачити.»

# Вправи

## quiz: Формальний і неформальний регістри

> Виберіть правильну форму.

1. Розмова з начальником: «___ мені допомогти?»
   - [ ] Можеш
   - [x] Чи могли б Ви
   > Формальне звертання до начальника.

2. SMS другу: «___! Де ти?»
   - [x] Привіт
   - [ ] Добрий день
   > Неформальне повідомлення.

3. Лист професору: «Шановний пане ___!»
   - [ ] Іван
   - [x] Іване Петровичу
   > Формальне звертання = ім'я + по батькові.

4. Закінчення офіційного листа:
   - [ ] Бувай!
   - [x] З повагою,
   > Офіційна формула.

5. Перехід на «ти» пропонує:
   - [x] Старший молодшому
   - [ ] Молодший старшому
   > Традиційне правило етикету.

# Підсумок

## Система «ти/Ви»

| Ти | Ви |
|----|-----|
| Друзі, родина | Незнайомці, старші |
| Неформальні ситуації | Офіційні ситуації |
| Ім'я | Пане/Пані + ім'я |
| 2 ос. однини | 2 ос. множини |

## Ключові формули

| Неформальний | Формальний |
|--------------|------------|
| Привіт! | Добрий день! |
| Дай... | Чи могли б Ви... |
| Дякую! | Дякую Вам! |
| Бувай! | До побачення! |

> **Головне**
>
> - Вибір «ти/Ви» залежить від віку, статусу, ситуації
> - «Ви» пишеться з великої літери в листах
> - Перехід на «ти» пропонує старший
> - Кожен регістр має свої лексичні маркери
