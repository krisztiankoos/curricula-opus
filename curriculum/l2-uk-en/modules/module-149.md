# Зміст уроку

## Розминка

### Що таке офіційно-діловий стиль?

**Офіційно-діловий стиль** — мова документів, законів, контрактів.

**Характеристики:**
- Стандартизованість
- Безособовість
- Точність формулювань
- Кліше та шаблони
- Відсутність емоцій

**Де вживається:**
- Заяви, довідки, накази
- Офіційні листи
- Договори, контракти
- Закони, постанови
- Протоколи

## Презентація

### Типи офіційних документів

| Тип | Приклад |
|-----|---------|
| Заява | Прошу надати відпустку... |
| Довідка | Довідка видана... |
| Наказ | Наказую звільнити... |
| Протокол | Слухали... Ухвалили... |
| Договір | Сторони домовилися... |
| Скарга | Прошу розглянути скаргу... |

### Структура заяви

```
                                        Директору ТОВ «Компанія»
                                        Петренку І.М.
                                        від менеджера відділу продажу
                                        Коваленко О.П.

                            ЗАЯВА

Прошу надати мені щорічну відпустку з 01.07.2024 по 14.07.2024
тривалістю 14 календарних днів.

25.06.2024                              _________ О.П. Коваленко
                                        (підпис)
```

### Стандартні формули

**Початок:**

| Формула | Вживання |
|---------|----------|
| Прошу... | заяви, прохання |
| Повідомляю... | інформування |
| Звертаюся з проханням... | листи |
| Відповідно до... | посилання на закон |
| У зв'язку з... | пояснення причини |

**Середина:**

| Формула | Приклад |
|---------|---------|
| відповідно до | відповідно до статті 5 |
| у зв'язку з | у зв'язку з хворобою |
| з метою | з метою покращення |
| на підставі | на підставі закону |
| згідно з | згідно з договором |

**Кінець:**

| Формула | Приклад |
|---------|---------|
| Прошу розглянути... | Прошу розглянути мою заяву. |
| Очікую на відповідь... | Очікую на Вашу відповідь. |
| Заздалегідь вдячний(-а)... | Заздалегідь вдячна. |
| З повагою... | З повагою, О. Петренко |

### Кліше офіційного стилю

| Кліше | Значення |
|-------|----------|
| вищезазначений | mentioned above |
| нижчепідписаний | the undersigned |
| в установленому порядку | in due course |
| належним чином | properly |
| за наявності | if available |
| за відсутності | in the absence of |
| мати на увазі | to keep in mind |
| взяти до уваги | to take into account |
| довести до відома | to inform |

### Офіційний лист

**Структура:**

1. **Шапка** (адресат, відправник)
2. **Тема** (не завжди)
3. **Звертання** (Шановний пане/пані!)
4. **Вступ** (мета листа)
5. **Основна частина** (деталі)
6. **Завершення** (очікування, подяка)
7. **Підпис** (посада, ім'я)

**Приклад:**

```
                                        Директору ТОВ «Партнер»
                                        Шевченку В.І.

Шановний пане Шевченко!

Звертаюся до Вас з приводу укладання договору
про співпрацю між нашими компаніями.

Відповідно до попередніх домовленостей, пропонуємо
розпочати співпрацю з 01.08.2024. Умови договору
надсилаємо у додатку.

Прошу повідомити про Ваше рішення до 15.07.2024.

З повагою,
Директор ТОВ «Компанія»
_________________ І.М. Петренко
```

### Заява на відпустку

**Приклад:**

```
Прошу надати мені щорічну оплачувану відпустку
з 01.07.2024 по 28.07.2024 тривалістю 28 календарних днів.
```

**Типи відпусток:**
- щорічна оплачувана відпустка
- відпустка без збереження заробітної плати
- навчальна відпустка
- відпустка по догляду за дитиною

### Скарга

**Структура:**

1. Адресат (кому)
2. Заявник (від кого)
3. Суть скарги (що трапилось)
4. Вимоги (чого хочете)
5. Додатки (докази)

**Приклад:**

```
                                        Директору магазину «Техніка»
                                        від покупця Коваленко О.П.
                                        тел.: +380 XX XXX XX XX

                            СКАРГА

15.06.2024 я придбав у Вашому магазині телевізор Samsung
(чек № 12345). Через тиждень виявилася несправність екрана.

Відповідно до Закону України «Про захист прав споживачів»,
прошу:
1. Замінити товар на аналогічний належної якості.
2. Або повернути кошти у повному обсязі.

Додатки:
1. Копія чека.
2. Гарантійний талон.
3. Фото несправності.

22.06.2024                              _________ О.П. Коваленко
```

### Безособові конструкції

В офіційному стилі переважають **безособові** форми:

| Особова | Безособова |
|---------|------------|
| Я прошу | Просимо / Прошу |
| Ми наказуємо | Наказуємо / Наказано |
| Я підписав | Підписано |
| Вони ухвалили | Ухвалено |

### Канцеляризми: використовувати обережно

**Надмірні канцеляризми** погіршують текст:

| ❌ Занадто складно | ✓ Краще |
|-------------------|---------|
| здійснити оплату | оплатити |
| провести перевірку | перевірити |
| надати допомогу | допомогти |
| мати місце | бути, траплятися |
| в даний час | зараз |

## Практика

### Напишіть заяву

**Ситуація:** Ви хочете взяти 3 дні відпустки без збереження зарплати.

**Шаблон:**
```
                                        Директору [компанія]
                                        [ПІБ директора]
                                        від [посада]
                                        [Ваше ПІБ]

                            ЗАЯВА

Прошу надати мені відпустку без збереження заробітної плати
з ___ по ___ тривалістю ___ календарних дні(-в) у зв'язку з ___.

[Дата]                                  _________ [Підпис, ПІБ]
```

### Напишіть офіційний лист

**Ситуація:** Ви хочете отримати довідку з університету.

**Шаблон:**
```
                                        Декану факультету [назва]
                                        [університет]
                                        від студента групи [номер]
                                        [Ваше ПІБ]

Шановний пане/пані [ім'я]!

Звертаюся до Вас з проханням видати довідку про те, що я
є студентом(-кою) [курс] курсу [факультет].

Довідка потрібна для подання до [куди].

Заздалегідь вдячний(-а).

[Дата]                                  _________ [Підпис, ПІБ]
```

### Виправте стиль

Зробіть речення більш офіційними:

1. Хочу відпустку.
   → **Прошу надати мені відпустку.**

2. Мені треба довідка.
   → **Прошу видати мені довідку.**

3. Телевізор зламався, поверніть гроші!
   → **Прошу повернути кошти за несправний товар.**

# Вправи

## quiz: Офіційно-діловий стиль

> Виберіть правильну формулу.

1. Початок заяви:
   - [x] Прошу надати...
   - [ ] Хочу отримати...
   > Офіційна формула.

2. Посилання на закон:
   - [ ] Як сказано в законі...
   - [x] Відповідно до закону...
   > Офіційне кліше.

3. Пояснення причини:
   - [x] У зв'язку з...
   - [ ] Тому що...
   > Офіційний сполучник.

4. Закінчення листа:
   - [ ] Бувайте здорові!
   - [x] З повагою,
   > Офіційна формула.

5. Замість «зараз» в офіційному стилі:
   - [ ] тепер
   - [x] на даний момент
   > Офіційне кліше (хоча «зараз» також допустимо).

# Підсумок

## Структура офіційних документів

| Елемент | Функція |
|---------|---------|
| Адресат | Кому |
| Заявник | Від кого |
| Назва | ЗАЯВА, СКАРГА |
| Текст | Суть справи |
| Дата, підпис | Засвідчення |

## Ключові формули

| Функція | Формула |
|---------|---------|
| Прохання | Прошу надати/видати... |
| Причина | У зв'язку з... |
| Посилання | Відповідно до... |
| Завершення | З повагою, |

> **Головне**
>
> - Офіційний стиль = стандартність + точність
> - Використовуйте кліше та шаблони
> - Безособові конструкції переважають
> - Уникайте надмірних канцеляризмів
