warm-up

You know how to read (—á–∏—Ç–∞—Ç–∏) and ask (–ø–∏—Ç–∞—Ç–∏).
But how do you do (—Ä–æ–±–∏—Ç–∏) or speak (–≥–æ–≤–æ—Ä–∏—Ç–∏)?

These are Class II verbs.
Class I likes -E- (—á–∏—Ç–∞—î—à).
Class II likes -I- (–≥–æ–≤–æ—Ä–∏—à).

It is a small difference, but it makes you sound fluent.

presentation

Class II Conjugation (-–∏—Ç–∏)

The pattern is predictable.

–ì–æ–≤–æ—Ä–∏—Ç–∏ (To speak)

  • –Ø –≥–æ–≤–æ—Ä—é (I speak) - ends in -—é (like Class I)
  • –¢–∏ –≥–æ–≤–æ—Ä–∏—à (You speak) - -–∏—à (Class I was -–µ—à)
  • –í—ñ–Ω –≥–æ–≤–æ—Ä–∏—Ç—å (He speaks) - -–∏—Ç—å (Class I was -–µ)
  • –ú–∏ –≥–æ–≤–æ—Ä–∏–º–æ (We speak) - -–∏–º–æ (Class I was -–µ–º–æ)
  • –í–∏ –≥–æ–≤–æ—Ä–∏—Ç–µ (You speak pl.) - -–∏—Ç–µ (Class I was -–µ—Ç–µ)
  • –í–æ–Ω–∏ –≥–æ–≤–æ—Ä—è—Ç—å (They speak) - -—è—Ç—å (Class I was -—é—Ç—å)

üí° The Trick

Class I = E (—á–∏—Ç–∞–µ—à) -> They = –Æ (—á–∏—Ç–∞—é—Ç—å).
Class II = I (–≥–æ–≤–æ—Ä–∏—à) -> They = –Ø (–≥–æ–≤–æ—Ä—è—Ç—å).

E -> U
I -> A

Key Verbs

Memorize these "Big 4":

  1. –†–æ–±–∏—Ç–∏ (To do / make) -> –Ø —Ä–æ–±–ª—é, –¢–∏ —Ä–æ–±–∏—à.
  2. –ë–∞—á–∏—Ç–∏ (To see) -> –Ø –±–∞—á—É, –¢–∏ –±–∞—á–∏—à.
  3. –õ—é–±–∏—Ç–∏ (To love / like) -> –Ø –ª—é–±–ª—é, –¢–∏ –ª—é–±–∏—à.
  4. –•–æ—Ç—ñ—Ç–∏ (To want) -> –Ø —Ö–æ—á—É, –¢–∏ —Ö–æ—á–µ—à. (Wait! This one is weird).

‚ö†Ô∏è Irregular: –•–æ—Ç—ñ—Ç–∏

It is a mix!
Singular uses -—á–µ- (Class I style): –¢–∏ —Ö–æ—á–µ—à, –í—ñ–Ω —Ö–æ—á–µ.

Actually, –•–æ—Ç—ñ—Ç–∏ behaves mostly like Class I in endings (-–µ—à, -–µ, -—É—Ç—å), but stems change.
Just memorize: –Ø —Ö–æ—á—É, –¢–∏ —Ö–æ—á–µ—à, –í–æ–Ω–∏ —Ö–æ—á—É—Ç—å.
