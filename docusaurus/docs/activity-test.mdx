# Activity Format Reference

This page demonstrates all **13 interactive activity types** used in curriculum modules.

---

## 1. Quiz (Multiple Choice)

  <QuizQuestion
    question="What does 'привіт' mean?"

  />
  <QuizQuestion
    question="Which is the feminine form of 'this'?"
    options={[`цей`, `ця`, `це`, `ці`]}
    correctIndex={1}

  <QuizQuestion
    question="How do you say 'thank you'?"

## 8. Error-Correction (Find and Fix) - A2+

<ErrorCorrection>
  <ErrorCorrectionItem
    sentence="Я бачу студент у бібліотеці."
    errorWord="студент"
    correctForm="студента"
    options={[`студент`, `студента`, `студенту`, `студентом`]}
    explanation="Animate masculine accusative = genitive form"
  />
  <ErrorCorrectionItem
    sentence="Вона читав книгу вчора."
    errorWord="читав"
    correctForm="читала"
    options={[`читав`, `читала`, `читало`, `читали`]}
    explanation="Past tense agrees with subject gender (feminine = -ла)"
  />
</ErrorCorrection>

## 9. Select (Multiple Correct Answers) - A2+

<Select>
  <SelectQuestion
    question="Which are valid accusative forms for 'книга'?"
    options={[`книгу`, `книги`, `книзі`, `книгою`]}
    correctAnswers={[`книгу`]}
    explanation="Only книгу is the accusative singular."
  />
  <SelectQuestion
    question="Which verbs are Class I conjugation?"
    options={[`читати`, `писати`, `говорити`, `любити`]}
    correctAnswers={[`читати`, `писати`]}
    explanation="читати and писати end in -ати (Class I), говорити/любити are Class II (-ити)"
  />
</Select>

## Summary

All **13 activity types** are now interactive:

| # | Activity | Level | Description |
|---|----------|-------|-------------|
| 1 | quiz | A1+ | Multiple choice (single answer) |
| 2 | match-up | A1+ | Pair matching |
| 3 | fill-in | A1+ | Gap fill with dropdown options |
| 4 | true-false | A1+ | True/false statements |
| 5 | anagram | A1 only | Letter unscrambling |
| 6 | unjumble | A1+ | Word reordering |
| 7 | group-sort | A1+ | Category sorting |
| 8 | error-correction | A2+ | Find and fix mistakes |
| 9 | select | A2+ | Multiple correct answers |
| 10 | translate | A2+ | Translation choice |
| 11 | cloze | A2+ | Passage with multiple blanks |
| 12 | dialogue-reorder | A2+ | Conversation ordering |
| 13 | mark-the-words | A2+ | Word identification |

**Note:** `observe` is a content callout, not an activity. Use it inline in lesson content for pattern discovery before grammar explanations.
