# Зміст уроку

## Розминка

### Повторення B1

У B1 ви вивчили основи пасивних дієприкметників:

| Дієслово | Дієприкметник | Приклад |
|----------|---------------|---------|
| написати | написан**ий** | написаний лист |
| відкрити | відкрит**ий** | відкрите вікно |
| зробити | зроблен**ий** | зроблена робота |

Сьогодні ми поглибимо ці знання.

## Презентація

### Утворення: повний огляд

**Суфікс -ний/-ений/-аний** (більшість дієслів):

| Тип дієслова | Схема | Приклад |
|--------------|-------|---------|
| -ити | -ений | зробити → зроблений |
| -ати | -аний | написати → написаний |
| -увати | -ований | організувати → організований |
| -ювати | -йований | досліджувати → досліджуваний |

**Суфікс -тий** (дієслова з голосною перед -ти):

| Дієслово | Дієприкметник |
|----------|---------------|
| відкрити | відкритий |
| закрити | закритий |
| забути | забутий |
| взяти | взятий |
| почати | початий |
| розбити | розбитий |

### Чергування приголосних

При утворенні дієприкметників відбуваються чергування:

| Чергування | Дієслово | Дієприкметник |
|------------|----------|---------------|
| д → дж | ходити → походити | походжений |
| т → ч | платити → заплатити | заплачений |
| с → ш | просити → попросити | попрошений |
| з → ж | возити → привезти | привезений |
| к → ч | пекти → спекти | спечений |
| г → ж | берегти → зберегти | збережений |

**Приклади в контексті:**
- Гроші **заплачені**. (платити → заплачений)
- Торт **спечений** бабусею. (пекти → спечений)
- Документи **збережені** в архіві. (берегти → збережений)

### Складні випадки

**Дієслова на -сти/-зти:**

| Дієслово | Дієприкметник |
|----------|---------------|
| принести | принесений |
| привезти | привезений |
| прочитати | прочитаний |
| спасти | спасенний |
| знайти | знайдений |

**Дієслова з подвоєнням:**

| Дієслово | Дієприкметник |
|----------|---------------|
| з'єднати | з'єднаний |
| здійснити | здійснений |
| виконати | виконаний |

### Дієприкметники як прикметники

Багато дієприкметників стали звичайними прикметниками:

| Дієприкметник | Як прикметник | Значення |
|---------------|---------------|----------|
| відкритий | відкрита людина | open (personality) |
| закритий | закрите суспільство | closed (society) |
| зайнятий | зайнята людина | busy (person) |
| освічений | освічена людина | educated |
| вихований | вихована дитина | well-mannered |
| організований | організована людина | organized |

### Узгодження в реченні

Дієприкметники узгоджуються з іменником:

| Відмінок | Чоловічий | Жіночий | Середній | Множина |
|----------|-----------|---------|----------|---------|
| Називний | написаний | написана | написане | написані |
| Родовий | написаного | написаної | написаного | написаних |
| Давальний | написаному | написаній | написаному | написаним |
| Знахідний | написаний/-ого | написану | написане | написані/-их |
| Орудний | написаним | написаною | написаним | написаними |
| Місцевий | написаному | написаній | написаному | написаних |

**Приклади:**
- Я читаю **написану** (Зн.) книгу.
- Він говорить про **написану** (Зн.) статтю.
- Ми задоволені **написаною** (Ор.) роботою.

### Дієприкметникові звороти

**Перед іменником:**
- **Написана Шевченком** поезія відома всім.
- **Збудований у XIX столітті** палац став музеєм.

**Після іменника:**
- Поезія, **написана Шевченком**, відома всім.
- Палац, **збудований у XIX столітті**, став музеєм.

> ⚡ **Стилістична примітка**
>
> Дієприкметникові звороти — ознака **книжного стилю**.
> У розмовній мові частіше вживають підрядні речення:
> - Книжне: Написана автором книга...
> - Розмовне: Книга, яку написав автор...

## Практика

### Утворіть дієприкметники

1. прочитати → **прочитаний**
2. зробити → **зроблений**
3. відкрити → **відкритий**
4. забути → **забутий**
5. заплатити → **заплачений**
6. спекти → **спечений**
7. принести → **принесений**
8. організувати → **організований**
9. з'єднати → **з'єднаний**
10. зберегти → **збережений**

### Вставте правильну форму

1. Я бачив ___ (побудувати) міст.
   → **побудований**

2. Вона читає ___ (перекласти) роман.
   → **перекладений**

3. Ми живемо в ___ (відремонтувати) квартирі.
   → **відремонтованій**

4. Це ___ (визнати) у світі вчений.
   → **визнаний**

5. Документи ___ (підписати) директором.
   → **підписані**

### Трансформуйте

Замініть підрядне речення дієприкметниковим зворотом:

1. Книга, яку написав Франко, стала класикою.
   → **Написана Франком** книга стала класикою.

2. Проблема, яку обговорили на засіданні, вирішена.
   → **Обговорена на засіданні** проблема вирішена.

3. Закон, який ухвалив парламент, набув чинності.
   → **Ухвалений парламентом** закон набув чинності.

## Продукція

### Опис досягнень

Опишіть досягнення, використовуючи дієприкметники:

**Приклад:**
«У нашому місті **збудовано** новий міст. **Відремонтовані** дороги полегшують рух. **Відкриті** нові школи приймають учнів. **Організовані** заходи приваблюють туристів.»

| -ити | -ений | зроблений |

## Чергування

| Звук | Приклад |
|------|---------|
| т → ч | заплачений |
| к → ч | спечений |
| д → дж | походжений |
| г → ж | збережений |

> **Головне**
>
> - Дієприкметники узгоджуються з іменником
> - Чергування приголосних — важлива особливість
> - Дієприкметникові звороти — книжний стиль
> - Багато дієприкметників стали прикметниками
