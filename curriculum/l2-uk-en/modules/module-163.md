### Приклад тексту

**Бізнес-новина:**
«Українська IT-компанія залучила 10 мільйонів доларів інвестицій від міжнародного фонду. За словами засновника, кошти буде спрямовано на розширення команди та вихід на нові ринки. Компанія планує подвоїти оборот протягом наступного року.»

### Сучасний контекст

**Економіка України:**

| Факт | Дані |
|------|------|
| Валюта | Гривня (₴, UAH), введена 1996 року |
| НБУ | Національний банк України — центральний банк |
| IT-сектор | Один із найбільших в Європі |
| Аграрний сектор | Україна — один із найбільших експортерів зерна |

**Історія гривні:**
- 1996: Введення гривні замість купоно-карбованця
- 2014-2015: Девальвація через кризу
- 2022+: Фіксований курс під час війни

**Приклади сучасних новин:**

1. «НБУ **зберіг** облікову ставку на рівні 25%.»

2. «Експорт зерна через «зерновий коридор» **відновлено**.»

3. «IT-сектор **залишається** одним із найбільших платників податків.»

4. «Міжнародні партнери **виділили** Україні черговий транш фінансової допомоги.»

5. «Уряд **запровадив** нові податкові пільги для бізнесу в умовах воєнного стану.»

## Практика

### Заповніть пропуски

1. Курс гривні ___ (зростати/падати) на 2%.
   → **зріс / впав**

2. Компанія ___ (залучити) інвестиції.
   → **залучила**

3. Банк ___ (кредитувати) малий бізнес.
   → **кредитує**

4. Сторони ___ (підписати) контракт.
   → **підписали**

5. Економіка ___ (зростати) третій квартал поспіль.
   → **зростає**

### Визначте термін

1. Загальна вартість усіх товарів та послуг країни → **ВВП**
2. Зростання цін → **інфляція**
3. Гроші, позичені в банку → **кредит**
4. Частка власності компанії → **акція**
5. Різниця між доходами та витратами (позитивна) → **прибуток**

### Перекладіть

1. The exchange rate fell by 5%.
   → **Обмінний курс впав на 5%.**

2. The company made a profit.
   → **Компанія отримала прибуток.**

3. We agreed on the terms.
   → **Ми домовилися про умови.**

# Вправи

## quiz: Економіка та бізнес

> Виберіть правильну відповідь.

1. «ВВП» означає:
   - [x] валовий внутрішній продукт
   - [ ] валютний внутрішній продукт
   > GDP = Gross Domestic Product.

2. «Інфляція» — це:
   - [x] зростання цін
   - [ ] падіння цін
   > Інфляція = зростання цін.

3. «Акція» — це:
   - [x] частка власності компанії
   - [ ] банківський рахунок
   > Акція = stock, share.

4. «Залучити інвестиції» означає:
   - [x] отримати гроші від інвесторів
   - [ ] інвестувати гроші
   > Залучити = to attract, raise.

5. ТОВ — це:
   - [x] товариство з обмеженою відповідальністю
   - [ ] товариство оптової торгівлі
   > LLC = Limited Liability Company.

6. **[Інтеграція: пасив + регістр]** Офіційний звіт НБУ: «Облікову ставку ___»
   - [x] збережено
   - [ ] зберегли
   - [ ] зберігається
   > Офіційний документ → -но/-то форма.

7. **[Інтеграція: пасив + регістр]** Розмова з колегою: «Мені ___, що компанія збанкрутувала.»
   - [ ] повідомлено
   - [x] сказали
   - [ ] говориться
   > Розмовний стиль → 3-я особа множини.

## Бізнес-лексика

| Термін | Значення |
|--------|----------|
| прибуток | дохід мінус витрати |
| інвестиція | вкладення грошей |
| акція | частка власності |
| угода | домовленість |

> **Головне**
>
> - ВВП = валовий внутрішній продукт
> - Інфляція = зростання цін
> - Прибуток vs Збиток
> - Переговори → Угода → Контракт
