## Gap 1: Vocative Case

### Status: NEEDS EXPLICIT MODULE

### Problem
The Vocative case is only briefly mentioned in A2 Module 38 (Case Integration) but never explicitly taught with its own module.

### Importance
- 7th case in Ukrainian - essential for direct address
- Used constantly in everyday speech: "Мамо!", "Іване!", "Друзі!"
- Different endings than other cases
- Cultural significance in Ukrainian communication

### Fix: Add to A1.3 (Modules 21-30)

**Recommended placement:** Insert as **Module 25A** or integrate into Module 29 (Family)

**Proposed Module: The Vocative - Calling Names**

**Grammar:**
- Vocative case formation for direct address
- Used when calling someone or addressing them directly

**Formation:**
| Gender | Nominative | Vocative | Pattern |
|--------|-----------|----------|---------|
| Masculine (hard) | Іван | Іване | + е |
| Masculine (soft) | Андрій | Андрію | й → ю |
| Masculine -ко | Петро | Петре | о → е |
| Feminine -а | мама | мамо | а → о |
| Feminine -я | Марія | Маріє | я → є |
| Feminine -ь | любов | любове | ь → е |

**Key patterns:**
- Мамо, де ти? (Mom, where are you?)
- Іване, йди сюди! (Ivan, come here!)
- Друзі, слухайте! (Friends, listen!)
- Боже мій! (My God!)

**Vocabulary (15 words):**
- Common vocative forms: мамо, тату, бабусю, дідусю, сину, дочко, друже, подруго, пане, пані, Боже, Господи, діти, люди, хлопці

**Activities:**
1. Transform: Nominative → Vocative
2. Gap-fill: Complete greetings
3. Role-play: Calling family members

## Gap 2: Ordinal Numbers

### Status: NEEDS EXPLICIT COVERAGE

### Problem
Cardinals (one, two, three) are taught in A1 Module 09, but ordinals (first, second, third) are not explicitly covered.

### Importance
- Essential for dates, floors, order, rankings
- Ordinals decline like adjectives (agreement required)
- Used constantly: "перший поверх", "друга година", "третє питання"

### Fix: Add to A1.3 or A2.1

**Recommended placement:** Integrate into Module 23 (Time/Calendar) or create new module in A2.1

**Proposed content for Module 23 expansion:**

**Grammar: Ordinal Numbers**

**Formation:**
| Cardinal | Ordinal (m/f/n) |
|----------|-----------------|
| один | перший/перша/перше |
| два | другий/друга/друге |
| три | третій/третя/третє |
| чотири | четвертий/четверта/четверте |
| п'ять | п'ятий/п'ята/п'яте |
| шість | шостий/шоста/шосте |
| сім | сьомий/сьома/сьоме |
| вісім | восьмий/восьма/восьме |
| дев'ять | дев'ятий/дев'ята/дев'яте |
| десять | десятий/десята/десяте |

**Key patterns:**
- Перший урок починається о дев'ятій. (First lesson starts at 9.)
- Я живу на третьому поверсі. (I live on the third floor.)
- Сьогодні двадцять перше серпня. (Today is August 21st.)

**Vocabulary (20 words):**
- Ordinals 1st-10th (all gender forms)
- Ordinal markers: останній, наступний, попередній

**Activities:**
1. Match-up: Cardinal → Ordinal
2. Gap-fill: Floor numbers, dates
3. Quiz: What order?

## Gap 3: Adverb Comparison

### Status: NEEDS EXPLICIT COVERAGE

### Problem
Adjective comparison is taught in A2 Modules 45-47, but adverb comparison (faster, more quietly) is not explicitly covered.

### Importance
- Adverbs compare differently than adjectives
- Many adverbs derive from adjectives but have distinct comparative forms
- Essential for natural expression: "говорити швидше", "працювати краще"

### Fix: Add to A2.2 after Module 47

**Recommended placement:** Add as **Module 47A** or expand Module 47

**Proposed content:**

**Grammar: Comparing Adverbs**

**Formation:**
- Most adverbs: add -іше/-ше to stem
- швидко → швидше (faster)
- повільно → повільніше (more slowly)
- добре → краще (better)
- погано → гірше (worse)

**Adverb vs Adjective:**
| Adjective | Adverb | Comparative Adverb |
|-----------|--------|-------------------|
| швидкий | швидко | швидше |
| повільний | повільно | повільніше |
| добрий | добре | краще |
| поганий | погано | гірше |
| багатий | багато | більше |
| малий | мало | менше |

**Key patterns:**
- Він біжить швидше за мене. (He runs faster than me.)
- Вона працює краще. (She works better.)
- Говори тихіше! (Speak more quietly!)
- Я хочу їсти більше. (I want to eat more.)

**Vocabulary (15 words):**
- Comparative adverbs: швидше, повільніше, краще, гірше, більше, менше, тихіше, голосніше, раніше, пізніше, ближче, далі, вище, нижче, глибше

**Activities:**
1. Transform: Adverb → Comparative
2. Gap-fill: Compare actions
3. Quiz: Adjective comparative vs Adverb comparative

## Gap 4: Aspect with Imperatives

### Status: MINOR GAP

### Problem
Imperative mood is taught in A2, but the crucial aspect distinction in imperatives is not emphasized.

### Importance
- Imperfective imperative = polite request, process
- Perfective imperative = single action, result expected
- Wrong aspect can sound rude or strange

**Example:**
- Читай! (Read! - process, habitual)
- Прочитай! (Read it! - complete this)
- Сідай! (Sit down! - polite, process)
- Сядь! (Sit! - abrupt, one action)

### Fix: Add note to B1 Module 75 (Advanced Aspect)

Add section on "Aspect in Imperatives" explaining:
- Imperfective = invitation, repeated, polite
- Perfective = specific task, result expected
- Negative imperatives usually imperfective: Не читай! (Don't read!)
