# Зміст уроку

## Розминка

### Яка різниця?

Прочитайте:
1. «Він не прийшов, бо захворів.»
2. «Він не прийшов, оскільки захворів.»
3. «Він не прийшов, через те що захворів.»

Чи однакові ці речення?

## Презентація

### Причинові сполучники (каузальні)

#### Базові

| Сполучник | Стиль | Приклад |
|-----------|-------|---------|
| бо | розмовний | Не прийшов, бо захворів. |
| тому що | нейтральний | Не прийшов, тому що захворів. |
| оскільки | книжний | Оскільки він захворів, не прийшов. |

**Правила:**
- «Бо» — після головного речення
- «Тому що» — після головного речення
- «Оскільки» — може бути на початку

#### Книжні / Офіційні

| Сполучник | Стиль | Приклад |
|-----------|-------|---------|
| через те що | офіційний | Через те що були перешкоди... |
| завдяки тому що | позитивна причина | Завдяки тому що він допоміг... |
| внаслідок того що | офіційний | Внаслідок того що... |
| у зв'язку з тим що | дуже офіційний | У зв'язку з тим що... |

**Приклади:**
1. «Через те що погода погіршилася, захід скасували.»
2. «Завдяки тому що ви допомогли, ми встигли.»
3. «У зв'язку з тим що звіт не готовий, засідання перенесено.»

### Причинові сполучники: відтінки

| Сполучник | Відтінок | Приклад |
|-----------|----------|---------|
| бо | нейтральна причина | Не їм, бо не голодний. |
| завдяки тому що | позитивна причина | Успіх завдяки тому, що... |
| через те що | негативна причина | Проблеми через те, що... |
| внаслідок того що | результат | Внаслідок того, що... |

### Допустові сполучники (концесивні)

#### Що таке допустовість?

Допустові сполучники виражають **протиріччя**:
«Хоча X, але Y» — Y відбувається всупереч X.

#### Базові

| Сполучник | Стиль | Приклад |
|-----------|-------|---------|
| хоча | нейтральний | Хоча він втомився, працював далі. |
| хоч | розмовний | Хоч і важко, але зробив. |
| хоча б | мінімум | Хоча б зателефонуй! |

**Приклади:**
1. «Хоча погода погана, ми підемо.»
2. «Хоч і не хотів, але погодився.»

#### Книжні / Офіційні

| Сполучник | Стиль | Приклад |
|-----------|-------|---------|
| незважаючи на те що | офіційний | Незважаючи на те що... |
| попри те що | книжний | Попри те що... |
| дарма що | розмовний | Дарма що обіцяв... |
| хай (нехай) | піднесений | Хай він і багатий, але... |

**Приклади:**
1. «Незважаючи на те що були труднощі, проєкт завершено.»
2. «Попри те що він молодий, він досвідчений.»
3. «Дарма що обіцяв — не зробив.»
4. «Нехай це й важко, але треба.»

### Порівняння причини та допустовості

| Тип | Логіка | Приклад |
|-----|--------|---------|
| Причина | X → Y | Не прийшов, бо захворів. |
| Допустовість | X, але Y | Хоча захворів, прийшов. |

### Позиція в реченні

#### Причинові

| Позиція | Сполучник | Приклад |
|---------|-----------|---------|
| Після | бо, тому що | Не прийшов, бо захворів. |
| Перед/Після | оскільки | Оскільки він захворів, не прийшов. |
| Перед | через те що | Через те що він захворів... |

#### Допустові

| Позиція | Сполучник | Приклад |
|---------|-----------|---------|
| Перед | хоча | Хоча він захворів, прийшов. |
| Перед/Після | незважаючи на те що | Прийшов, незважаючи на хворобу. |

### Пунктуація

**Правило:** Складнопідрядні речення зазвичай відокремлюються комою.

| Приклад | Пунктуація |
|---------|------------|
| Не прийшов, бо захворів. | ✓ кома |
| Оскільки він захворів, не прийшов. | ✓ кома |
| Хоча погода погана, підемо. | ✓ кома |

### Стилістична шкала

**Причинові:**
```
Розмовний ← бо ← тому що ← оскільки ← через те що ← у зв'язку з тим що → Офіційний
```

**Допустові:**
```
Розмовний ← хоч ← хоча ← попри те що ← незважаючи на те що → Офіційний
```

## Практика

### Виберіть правильний сполучник

1. Він запізнився, ___ були затори. (нейтрально)
   → **тому що / бо**

2. ___ він ще молодий, він дуже досвідчений. (допустовість)
   → **Хоча**

3. ___ погіршення погоди, захід скасовано. (офіційно)
   → **У зв'язку з / Через**

4. ___ він багато працював, не встиг. (допустовість)
   → **Хоча / Незважаючи на те що**

5. Успіх досягнуто ___ ваша підтримка. (позитивна причина)
   → **завдяки тому що / завдяки**

### Трансформуйте речення

1. «Він не прийшов, бо захворів.» → офіційний стиль
   → **«Він не прийшов у зв'язку з хворобою.»**

2. «Хоча важко, треба робити.» → офіційний стиль
   → **«Незважаючи на труднощі, необхідно виконати роботу.»**

3. «Через погану погоду...» → розмовний стиль
   → **«Бо погода погана...»**

### Перекладіть

1. He didn't come because he was sick.
   → **Він не прийшов, бо / тому що захворів.**

2. Although it's difficult, we'll try.
   → **Хоча це важко, ми спробуємо.**

3. Despite the problems, the project was completed.
   → **Незважаючи на проблеми, проєкт завершено.**

## quiz: Причинові та допустові сполучники

> Виберіть правильну відповідь.

1. «Бо» — це стиль:
   - [x] розмовний
   - [ ] офіційний
   > Найрозмовніший.

2. «Оскільки» може стояти:
   - [x] на початку речення
   - [ ] лише в кінці
   > Гнучка позиція.

3. «Хоча» виражає:
   - [x] допустовість
   - [ ] причину
   > Concession.

4. «Завдяки тому що» вказує на:
   - [x] позитивну причину
   - [ ] негативну причину
   > Thanks to.

5. «Незважаючи на те що» — це:
   - [x] офіційний допустовий
   - [ ] розмовний причиновий
   > Despite the fact that.

6. **[Інтеграція: пасив]** «___ звіт не готовий, засідання перенесено.»
   - [x] У зв'язку з тим що
   - [ ] Бо
   - [ ] Хоча
   > Офіційний стиль → офіційний сполучник.

7. **[Інтеграція: регістр]** Розмова: «Не прийшов, ___ захворів.»
   - [x] бо
   - [ ] оскільки
   - [ ] у зв'язку з тим що
   > Розмовний стиль → бо.

| Стиль | Сполучник |

> **Головне**
>
> - Причина: X → Y (бо, тому що)
> - Допустовість: X, але Y (хоча)
> - Стиль визначає вибір сполучника
> - Пунктуація: кома перед/після
