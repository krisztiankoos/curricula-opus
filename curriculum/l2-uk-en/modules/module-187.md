## Практика

### Заповніть пропуски

1. Компанія ___ (розробляти) нову версію застосунку.
   → **розробляє**

2. Система використовує ___ для аналізу даних.
   → **алгоритми / ШІ / машинне навчання**

3. Файл ___ (зашифрувати) для безпеки.
   → **зашифровано**

4. ___ допомагає захистити акаунт.
   → **Двофакторна автентифікація**

5. Дані ___ (зберігати) в хмарі.
   → **зберігаються**

### Визначте термін

1. Програма, яка імітує розумну поведінку → **ШІ / штучний інтелект**
2. Захист комп'ютерних систем від атак → **кібербезпека**
3. Місце зберігання даних в інтернеті → **хмара / хмарний сервіс**
4. Перетворення даних у нечитабельний формат → **шифрування**
5. Процес виявлення помилок у програмі → **тестування**

### Перекладіть

1. The company developed an AI-based application.
   → **Компанія розробила застосунок на основі ШІ.**

2. Data is encrypted for security.
   → **Дані зашифровано для безпеки.**

3. Update the software to the latest version.
   → **Оновіть програмне забезпечення до останньої версії.**

## quiz: Технології

> Виберіть правильну відповідь.

1. «ШІ» означає:
   - [x] штучний інтелект
   - [ ] широкий інтернет
   > Artificial intelligence.

2. «Машинне навчання» — це:
   - [x] тип ШІ
   - [ ] навчання машиністів
   > Machine learning.

3. «Шифрування» означає:
   - [x] захист даних кодуванням
   - [ ] видалення даних
   > Encryption.

4. «Фішинг» — це:
   - [x] шахрайство для отримання даних
   - [ ] тип риболовлі
   > Online scam.

5. «Хмарний сервіс» — це:
   - [x] зберігання даних в інтернеті
   - [ ] прогноз погоди
   > Cloud service.

6. **[Інтеграція: пасив]** Технічний звіт: «Систему ___ для підвищення безпеки.»
   - [x] оновлено
   - [ ] оновили
   - [ ] оновлюється
   > Офіційний стиль → -но/-то.

7. **[Інтеграція: регістр]** Розмова: «___ нову версію застосунку.»
   - [ ] Випущено
   - [x] Випустили
   - [ ] Випускається
   > Розмовний стиль → 3-я особа мн.
