### Приклад тексту

**Новина:**
«Верховна Рада України у другому читанні ухвалила законопроєкт про реформу судової системи. За документ проголосували 278 народних депутатів. Опозиція критикувала окремі положення закону, проте голосувала «за». Президент очікується підписати закон протягом тижня.»

1. «Україна **отримала** статус кандидата на членство в ЄС у червні 2022 року.»

2. «Верховна Рада **ухвалила** закон про мобілізацію після тривалих дебатів.»

3. «Уряд **запровадив** нові правила перетину кордону для чоловіків призовного віку.»

4. «Президент **підписав** указ про продовження воєнного стану.»

## Практика

### Заповніть пропуски

1. Верховна Рада ___ (ухвалити) новий закон.
   → **ухвалила**

2. На виборах громадяни ___ (голосувати) за кандидатів.
   → **голосують**

3. Президент ___ (призначити) нового міністра.
   → **призначив**

4. Опозиція ___ (критикувати) рішення уряду.
   → **критикує / критикувала**

5. Законопроєкт ___ (відхилити) у першому читанні.
   → **відхилили / було відхилено**

### Визначте термін

1. Орган, що приймає закони → **Верховна Рада**
2. Голова уряду → **Прем'єр-міністр**
3. Місце, де голосують → **виборча дільниця**
4. Документ, за який голосують → **законопроєкт**
5. Група депутатів однієї партії → **фракція**

### Перекладіть

1. The parliament passed the law.
   → **Парламент (Верховна Рада) ухвалив закон.**

2. The president vetoed the bill.
   → **Президент наклав вето на законопроєкт.**

3. Voter turnout was 65%.
   → **Явка виборців склала 65%.**

# Вправи

## quiz: Політика та державне управління

> Виберіть правильну відповідь.

1. Верховна Рада — це орган:
   - [x] законодавчої влади
   - [ ] виконавчої влади
   > Парламент приймає закони.

2. «Ухвалити закон» означає:
   - [x] прийняти закон
   - [ ] відхилити закон
   > Ухвалити = to adopt, pass.

3. Хто очолює Кабінет Міністрів?
   - [ ] Президент
   - [x] Прем'єр-міністр
   > Прем'єр — голова уряду.

4. «Фракція» — це:
   - [x] група депутатів однієї партії
   - [ ] виборча дільниця
   > Фракція = faction.

5. «Накласти вето» означає:
   - [x] заблокувати закон
   - [ ] підписати закон
   > Вето = блокування.

6. **[Інтеграція: пасив + регістр]** Офіційне повідомлення: «Закон ___»
   - [x] ухвалено
   - [ ] ухвалили
   - [ ] ухвалюється
   > Офіційний стиль → -но/-то форма.

7. **[Інтеграція: пасив + регістр]** Розмова з другом: «Мені ___, що закон прийняли.»
   - [ ] повідомлено
   - [x] сказали
   - [ ] говориться
   > Розмовний стиль → 3-я особа множини.
