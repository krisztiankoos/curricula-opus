# Зміст уроку

## Розминка

### Як ми насправді говоримо?

Порівняйте:

| Офіційний | Розмовний |
|-----------|-----------|
| Я не знаю. | Не знаю. / Не зна'ю. |
| Куди ти йдеш? | Куди йдеш? / Куди? |
| Це дуже добре. | Це круто! / Кльово! |
| Я думаю, що так. | Думаю, так. / Та да. |

**Розмовний стиль** — мова щоденного спілкування.

## Презентація

### Характеристики розмовного стилю

| Риса | Опис |
|------|------|
| Неповнота | Опущення слів |
| Спонтанність | Без підготовки |
| Емоційність | Вигуки, інтонація |
| Простота | Прості конструкції |
| Конкретність | Конкретні ситуації |

### Фонетичні особливості

| Явище | Приклад |
|-------|---------|
| Редукція | «не знаю» → «не зна'ю» |
| Злиття | «чого ти» → «чоти» |
| Скорочення | «мабуть» → «мабть» |

> ⚠️ **Увага**
>
> Це особливості *усного* мовлення. На письмі зберігаємо повні форми!

### Неповні речення

| Повне | Неповне |
|-------|---------|
| Куди ти йдеш? | Куди? |
| Що ти робиш? | Що робиш? |
| Коли ти прийдеш? | Коли? |
| Як твої справи? | Як справи? |
| Я теж так думаю. | Я теж. |

### Частки та вигуки

**Частки:**

| Частка | Функція | Приклад |
|--------|---------|---------|
| ну | вступ, роздум | Ну, не знаю... |
| от | вказівка | От бачиш! |
| та | підсилення | Та ні! |
| же / ж | підсилення | Ти ж знаєш! |
| оце | здивування | Оце так! |

**Вигуки:**

| Вигук | Емоція |
|-------|--------|
| Ого! | здивування |
| Ой! | переляк |
| Ех! | розчарування |
| Ага! | згода |
| Угу! | підтвердження |
| Овва! | здивування |
| Отакої! | несподіванка |

### Розмовна лексика

**Неформальні синоніми:**

| Нейтральний | Розмовний |
|-------------|-----------|
| добре | гаразд, окей |
| погано | погано, кепсько |
| їжа | їжа, їдло |
| гроші | гроші, бабки (сленг) |
| людина | людина, чувак (сленг) |
| працювати | працювати, пахати (сленг) |

**Зменшувальні форми:**

| Нейтральний | Зменшувальний |
|-------------|---------------|
| хвилина | хвилинка |
| трохи | трішки |
| тихо | тихенько |
| добре | добренько |

### Сленг: обережно!

**Сучасний молодіжний сленг:**

| Сленг | Значення | Походження |
|-------|----------|------------|
| круто | добре | давній сленг |
| кайф | задоволення | давній сленг |
| по кайфу | приємно | давній сленг |
| лайк | вподобання | англ. like |
| чілити | відпочивати | англ. chill |

> ⚠️ **Застереження**
>
> Сленг швидко змінюється і може звучати недоречно.
> Краще використовувати нейтральну розмовну лексику.

### Розмовний vs Неграмотний

| Розмовний ✓ | Неграмотний ✗ |
|-------------|---------------|
| Не знаю. | *Нє знаю. |
| Чого ти? | *Чого ти хоч? |
| Куди йдеш? | *Куда йдеш? |
| Дякую! | *Спасібо! |

> ⚠️ **Важливо**
>
> Розмовний стиль — це **стиль**, а не помилки!
> «Нє», «куда», «спасібо» — русизми, НЕ розмовний стиль.

### Типові конструкції

**Питання-підтвердження:**
- «Ти ж прийдеш?» (очікую «так»)
- «Правда?», «Так?», «Чи не так?»

**Емфатичні конструкції:**
- «Це ж треба!» (здивування)
- «От тобі й на!» (несподіванка)
- «Отакої!» (здивування)

**Повтори:**
- «Так-так-так» (підтвердження)
- «Ну-ну» (сумнів)
- «Добре-добре» (згода)

### Діалог: приклад

**Офіційна версія:**
— Добрий день! Чи не могли б ви мені допомогти?
— Звичайно. Чим можу бути корисним?
— Мені потрібна інформація про розклад.
— Зачекайте, будь ласка, я перевірю.

**Розмовна версія:**
— Привіт! Слухай, можеш допомогти?
— Ага, що треба?
— Та от, розклад хочу дізнатися.
— Зараз, секунду, гляну.

## Практика

### Трансформуйте

**Офіційний → Розмовний:**

1. Я не розумію, що ви маєте на увазі.
   → **Не розумію, про що ти.**

2. Чи не могли б ви зачекати?
   → **Почекай, добре?**

3. На жаль, я не маю змоги допомогти.
   → **Та ні, не можу, вибач.**

**Розмовний → Офіційний:**

1. Слухай, треба поговорити.
   → **Прошу знайти час для розмови.**

2. Не зна'ю, може, завтра.
   → **Не знаю. Можливо, завтра.**

3. Ну от, я ж казав!
   → **Саме про це я говорив.**

### Визначте стиль

1. «Відповідно до законодавства України...»
   → **Офіційний**

2. «Ну от, я й думаю, може, підемо?»
   → **Розмовний**

3. «Учора парламент ухвалив важливе рішення.»
   → **Публіцистичний**

4. «Ой, та ні, я зовсім забув!»
   → **Розмовний**

## Продукція

### Напишіть діалог

Напишіть короткий діалог (6-8 реплік) між друзями, використовуючи розмовний стиль:

**Тема:** Плани на вихідні

**Приклад початку:**

— Та не знаю ще. А що?
— Може, кудись підемо?
— О, гарна ідея! Куди?

# Вправи

## quiz: Розмовний стиль

> Визначте характеристику.

1. «Ну», «от», «та» — це:
   - [x] частки
   - [ ] вигуки
   > Частки вносять відтінки значення.

2. «Ого!», «Ой!», «Ех!» — це:
   - [ ] частки
   - [x] вигуки
   > Вигуки виражають емоції.

3. Яке слово є русизмом (НЕ розмовний стиль)?
   - [ ] не знаю
   - [x] спасібо
   > «Спасібо» — русизм, правильно «дякую».

4. «Куди?» замість «Куди ти йдеш?» — це:
   - [x] неповне речення
   - [ ] помилка
   > Неповнота типова для розмовного стилю.

5. «Трішки», «хвилинка» — це:
   - [x] зменшувальні форми
   - [ ] сленг
   > Демінутиви (зменшувальні) типові для розмовного стилю.

# Підсумок

## Розмовний стиль

| Риса | Приклад |
|------|---------|
| Неповнота | «Куди?» замість «Куди йдеш?» |
| Частки | «ну», «от», «та», «ж» |
| Вигуки | «Ого!», «Ой!», «Ех!» |
| Емоційність | Інтонація, повтори |

## Розмовний ≠ Неграмотний

| ✓ Розмовний | ✗ Русизм/помилка |
|-------------|------------------|
| Не знаю | *Нє знаю |
| Куди? | *Куда? |
| Дякую | *Спасібо |

> **Головне**
>
> - Розмовний стиль — це стиль, а не помилки
> - Неповні речення та частки — норма
> - Русизми — НЕ розмовний стиль
> - Сленг змінюється швидко, використовуйте обережно
