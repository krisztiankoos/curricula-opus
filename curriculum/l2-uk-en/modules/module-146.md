# Зміст уроку

## Розминка

### Чотири форми — один зміст?

Одна ідея — різні форми:

| Форма | Речення | Стиль |
|-------|---------|-------|
| Дієприкметник | Закон прийнятий. | Нейтральний |
| -но/-то | Закон прийнято. | Офіційний |
| -ся | Закон приймається. | Процес |
| 3-я ос. мн. | Закон прийняли. | Розмовний |

Як вибрати? Залежить від **контексту** та **стилю**.

## Презентація

### Алгоритм вибору

```
1. Який стиль тексту?
   ├── Офіційний → -но/-то
   ├── Науковий → -ся (процес) або дієприкметник
   ├── Розмовний → 3-я ос. мн.
   └── Нейтральний → дієприкметник

2. Що в фокусі?
   ├── Результат → дієприкметник
   ├── Процес → -ся
   ├── Факт (офіційно) → -но/-то
   └── Дія (хтось зробив) → 3-я ос. мн.
```

### Контекст 1: Офіційний документ

**Характеристики:**
- Протоколи, накази, закони
- Безособовість
- Формальність

**Типова форма: -но/-то**

**Приклад:**
«**Розглянуто** питання бюджету. **Ухвалено** рішення про фінансування. **Доручено** відділу підготувати звіт.»

### Контекст 2: Новини

**Характеристики:**
- Об'єктивність
- Факти без емоцій
- Стислість

**Типова форма: -но/-то або дієприкметник**

**Приклад:**
«Угоду **підписано** в Брюсселі. Документ **схвалений** обома сторонами. За інформацією, переговори **тривали** кілька місяців.»

### Контекст 3: Наукова стаття

**Характеристики:**
- Аналіз процесів
- Абстрактність
- Безособовість

**Типова форма: -ся (для процесів) або -но/-то (для фактів)**

**Приклад:**
«У статті **розглядається** проблема... Дані **аналізуються** за методом... **Встановлено**, що...»

### Контекст 4: Розмова

**Характеристики:**
- Особистість
- Природність
- Конкретність

**Типова форма: 3-я особа множини**

**Приклад:**
«Мені **сказали**, що зустріч перенесли. Його **запитали** про плани. Нам **показали** нову квартиру.»

### Контекст 5: Опис стану

**Характеристики:**
- Результат дії
- Статичність
- Характеристика

**Типова форма: дієприкметник**

**Приклад:**
«Вікно **відкрите**. Робота **зроблена**. Лист **написаний** і **відправлений**.»

### Змішані контексти

Реальні тексти часто змішують форми:

**Новинний репортаж:**
«Учора в Києві **підписано** важливу угоду *(офіційний факт)*. Документ **був підготовлений** протягом місяця *(опис процесу)*. Як **повідомили** у пресслужбі *(розмовний переказ)*, угода **набуде** чинності з наступного року.»

## Практика

### Визначте стиль і форму

1. «Засідання відкрито о 10:00.»
   - Стиль: **Офіційний**
   - Форма: **-но/-то**

2. «Будинок будується вже два роки.»
   - Стиль: **Нейтральний/інформативний**
   - Форма: **-ся** (процес)

3. «Мені сказали цікаву новину.»
   - Стиль: **Розмовний**
   - Форма: **3-я ос. мн.**

4. «Книга написана відомим автором.»
   - Стиль: **Нейтральний**
   - Форма: **Дієприкметник**

### Трансформуйте стиль

**Офіційний → Розмовний:**

1. Угоду підписано.
   → Угоду **підписали**.

2. Рішення прийнято одноголосно.
   → Рішення **прийняли** одноголосно.

3. Мене було повідомлено про зміни.
   → Мене **повідомили** про зміни.

**Розмовний → Офіційний:**

1. Документи перевірили.
   → Документи **перевірено**.

2. Заяву розглянули.
   → Заяву **розглянуто**.

3. Наказ підписали.
   → Наказ **підписано**.

### Виберіть правильну форму

**Ситуація 1: Протокол засідання**
- Питання ___ (розглянути).
  → **розглянуто**

**Ситуація 2: Розмова з другом**
- Мені ___ (сказати), що ти приїдеш.
  → **сказали**

**Ситуація 3: Наукова стаття**
- У роботі ___ (аналізувати) дані.
  → **аналізуються**

**Ситуація 4: Опис кімнати**
- Вікна ___ (помити), підлога ___ (прибрати).
  → **помиті**, **прибрана**

### Напишіть тексти різними стилями

**Тема:** Відкриття нового музею

**Офіційний:**
«Музей **відкрито** 15 травня. **Організовано** урочисту церемонію. **Запрошено** представників влади.»

**Новинний:**
«У Львові **відкрито** новий музей. Церемонія **відбулася** вчора. Музей **присвячений** історії міста.»

**Розмовний:**
«Уявляєш, у Львові **відкрили** новий музей! Мені **розповіли**, що там дуже цікаво. Нас **запросили** на відкриття.»

## Продукція

### Перепишіть текст

Перепишіть новинний текст розмовним стилем:

**Оригінал (новини):**
«Вчора було підписано угоду про співпрацю. Документ був підготовлений протягом місяця. Переговори проведено успішно.»

**Розмовний варіант:**
«Вчора **підписали** угоду про співпрацю. Документ **готували** цілий місяць. **Кажуть**, переговори **пройшли** успішно.»

# Вправи

## quiz: Вибір пасивної форми

> Виберіть найкращу форму для контексту.

1. Протокол засідання: «Рішення ___»
   - [ ] прийняте
   - [x] прийнято
   - [ ] прийняли
   > Офіційний документ → -но/-то

2. Розмова: «Мені ___ про це вчора.»
   - [ ] сказано
   - [x] сказали
   - [ ] говориться
   > Розмовний стиль → 3-я ос. мн.

3. Наукова стаття: «Проблема ___ у багатьох дослідженнях.»
   - [x] розглядається
   - [ ] розглянуто
   - [ ] розглянули
   > Науковий стиль, процес → -ся

4. Опис: «Двері ___, вікна ___.»
   - [x] закриті, відкриті
   - [ ] закрито, відкрито
   - [ ] закрили, відкрили
   > Опис стану → дієприкметник

5. Новини: «Угоду ___ у Брюсселі.»
   - [ ] підписана
   - [x] підписано
   - [ ] підписали
   > Новинний стиль → -но/-то

# Підсумок

## Вибір пасивної форми

| Контекст | Форма | Приклад |
|----------|-------|---------|
| Офіційний документ | -но/-то | Рішення прийнято. |
| Новини | -но/-то / дієприкм. | Угоду підписано. |
| Наукова стаття | -ся | Проблема розглядається. |
| Розмова | 3-я ос. мн. | Мені сказали. |
| Опис стану | Дієприкметник | Вікно відкрите. |

> **Головне**
>
> - Вибір форми залежить від стилю та контексту
> - Офіційний стиль → -но/-то
> - Розмовний стиль → 3-я особа множини
> - Науковий стиль (процес) → -ся
> - Опис результату → дієприкметник
