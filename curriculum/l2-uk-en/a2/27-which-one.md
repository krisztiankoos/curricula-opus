# Warm-up

How do you describe things with more detail? "The book **that I read**," "The person **who speaks**"? This module teaches relative clauses using **який** (which/who/that).

---

# Presentation

## Relative Pronoun який

**Який** agrees with the noun it refers to in gender and number, but its case depends on its role in the relative clause.

### Agreement in Gender/Number

| Noun Gender | Form of який |
|-------------|--------------|
| masculine | який |
| feminine | яка |
| neuter | яке |
| plural | які |

### Examples

- Людина, **яка** говорить... (The person who speaks...)
- Книга, **яку** я читаю... (The book that I'm reading...)
- Хлопець, **якому** я допомагаю... (The boy whom I'm helping...)

## Case in Relative Clause

| Role | Case | Example |
|------|------|---------|
| Subject | Nom. | Людина, яка говорить... |
| Direct object | Acc. | Книга, яку я читаю... |
| Indirect object | Dat. | Хлопець, якому я допомагаю... |
| With preposition | varies | Друг, з яким я говорю... |

---

# Summary

**Який** agrees with antecedent in gender/number.
**Case** depends on role in the relative clause.

## match-up: Relative Pronouns

- який :: which (m)
- яка :: which (f)
- яке :: which (n)
- які :: which (pl)
- якого :: of which (m/n gen)
- якої :: of which (f gen)
- якому :: to which (m/n dat)
- якій :: to which (f dat)
- яким :: with which (m/n instr)
- якою :: with which (f instr)
- якими :: with which (pl instr)
- яких :: of which (pl gen)
