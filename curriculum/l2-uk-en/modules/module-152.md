# Зміст уроку

## Розминка

### Один зміст — п'ять стилів

**Ідея:** Людина втратила роботу.

| Стиль | Формулювання |
|-------|--------------|
| Розмовний | Мене звільнили. От халепа! |
| Офіційний | Працівника звільнено згідно з наказом № 45. |
| Публіцистичний | Рівень безробіття зростає: тисячі людей втрачають роботу. |
| Науковий | Дослідження показує кореляцію між економічними кризами та рівнем безробіття. |
| Художній | Він стояв біля заводських воріт, тримаючи в руках папірець — останню платіжку за своє життя тут. |

## Презентація

### Алгоритм визначення стилю

```
1. Лексика
   ├── Кліше, канцеляризми → Офіційний
   ├── Терміни → Науковий
   ├── Метафори, епітети → Художній
   ├── Оцінна лексика → Публіцистичний
   └── Частки, вигуки → Розмовний

2. Граматика
   ├── -но/-то, безособові → Офіційний
   ├── -ся (процеси) → Науковий
   ├── Складні речення → Науковий/Художній
   └── Неповні речення → Розмовний

3. Контекст
   ├── Документ → Офіційний
   ├── Стаття, дослідження → Науковий
   ├── Новини, ЗМІ → Публіцистичний
   ├── Література → Художній
   └── Діалог, SMS → Розмовний
```

### Маркери стилів: зведена таблиця

| Стиль | Лексика | Граматика | Синтаксис |
|-------|---------|-----------|-----------|
| Розмовний | частки, вигуки | 3-я ос. мн. | неповні речення |
| Офіційний | кліше, канцеляризми | -но/-то | складні, стандартні |
| Науковий | терміни | -ся (процес) | складні, логічні |
| Публіцистичний | оцінна лексика | різні | риторичні питання |
| Художній | тропи (метафори) | різні | образні, ритмічні |

### Типові помилки змішування стилів

**❌ Офіційний + Розмовний:**
«Відповідно до закону, ну от, працівника звільнено.»
→ ✓ «Відповідно до закону, працівника звільнено.»

**❌ Науковий + Емоційний:**
«Дослідження показує жахливі результати!!!»
→ ✓ «Дослідження виявило значні відхилення від норми.»

**❌ Розмовний + Канцеляризми:**
«Ну, у зв'язку з вищезазначеним, я типу не прийду.»
→ ✓ «Слухай, не прийду, вибач.»

### Трансформація: Розмовний → Офіційний

| Розмовний | Офіційний |
|-----------|-----------|
| Мене звільнили. | Працівника звільнено. |
| Дай мені відпустку. | Прошу надати мені відпустку. |
| Ми домовилися. | Сторонами досягнуто домовленості. |
| Треба перевірити. | Необхідно провести перевірку. |
| Я згоден. | Погоджено. |

### Трансформація: Офіційний → Розмовний

| Офіційний | Розмовний |
|-----------|-----------|
| Угоду підписано. | Угоду підписали. |
| Прошу повідомити. | Скажи, будь ласка. |
| У зв'язку з хворобою | Бо захворів(-ла). |
| На даний момент | Зараз |
| Вищезазначений | Той, про якого говорили |

### Трансформація: Нейтральний → Публіцистичний

| Нейтральний | Публіцистичний |
|-------------|----------------|
| Відбулася зустріч. | Відбулася історична зустріч. |
| Парламент прийняв закон. | Парламент ухвалив довгоочікуваний закон. |
| Економіка змінюється. | Економіка на порозі змін. |
| Люди вийшли на вулицю. | Тисячі громадян вийшли на захист своїх прав. |

## Практика

### Визначте стиль

1. «Прошу розглянути питання щодо надання відпустки.»
   → **Офіційний** (кліше: «прошу розглянути», «щодо надання»)

2. «Ну от, я й думаю — може, підемо?»
   → **Розмовний** (частки: «ну», «от», неповнота)

3. «Хвиля протестів охопила країну.»
   → **Публіцистичний** (метафора: «хвиля», оцінність)

4. «У статті аналізуються фактори впливу на процес.»
   → **Науковий** (терміни, -ся пасив)

5. «Сонце сідало за обрій, фарбуючи небо в пурпур.»
   → **Художній** (образність, метафора)

### Виправте стилістичну помилку

1. ❌ «Прошу надати мені відпустку, бо типу хочу відпочити.»
   ✓ **«Прошу надати мені відпустку у зв'язку з сімейними обставинами.»**

2. ❌ «Дослідження показує, що це реально круто!»
   ✓ **«Дослідження виявило позитивні результати.»**

3. ❌ «Привіт! У зв'язку з вищезазначеним прошу зателефонувати.»
   ✓ **«Привіт! Зателефонуй мені, будь ласка.»**

### Трансформуйте текст

**Вихідний (розмовний):**
«Уявляєш, мене вчора на роботі отак взяли і звільнили! Кажуть — скорочення. Ну от, тепер шукаю нову роботу.»

**Офіційний:**
«Повідомляю, що 15.06.2024 мене було звільнено з посади у зв'язку зі скороченням штату. На даний момент перебуваю у пошуку нового місця роботи.»

**Публіцистичний:**
«Хвиля скорочень не оминає нікого. Учора ще один працівник став жертвою "оптимізації" — улюбленого слова сучасних роботодавців.»

### Напишіть три версії

**Тема:** Ви не можете прийти на зустріч.

**Розмовний (другу):**
«Привіт! Слухай, не зможу сьогодні. Щось термінове вискочило. Може, завтра?»

**Офіційний (керівнику):**
«Шановний пане директоре! На жаль, у зв'язку з непередбаченими обставинами не маю змоги бути присутнім(-ою) на сьогоднішній зустрічі. Прошу перенести зустріч на завтра.»

**Нейтральний (колезі):**
«Добрий день! На жаль, не зможу бути на сьогоднішній зустрічі. Чи можемо перенести на завтра?»

## Продукція

### Комплексна трансформація

Візьміть одну новину з інтернету та перепишіть її:
1. Розмовним стилем (як би ви розповіли другу)
2. Офіційним стилем (як офіційне повідомлення)

## quiz: Визначення стилю

> Визначте стиль тексту.

1. «Відповідно до п. 5 статті 12 Закону України...»
   - [x] офіційний
   - [ ] науковий
   > Кліше, посилання на закон.

2. «Ну, і що ти про це думаєш?»
   - [ ] публіцистичний
   - [x] розмовний
   > Частка «ну», пряме звертання.

3. «Чи зможе Україна подолати економічні виклики?»
   - [x] публіцистичний
   - [ ] науковий
   > Риторичне питання, актуальність.

4. «Місяць виплив із хмар, як срібна монета.»
   - [ ] публіцистичний
   - [x] художній
   > Порівняння, образність.

5. «У роботі досліджується вплив фактору X на процес Y.»
   - [ ] офіційний
   - [x] науковий
   > Терміни, -ся пасив.

# Підсумок

## Алгоритм визначення стилю

1. Шукайте лексичні маркери
2. Аналізуйте граматику
3. Враховуйте контекст

## Типові помилки

| ❌ Помилка | ✓ Виправлення |
|-----------|---------------|
| Змішування стилів | Один стиль на текст |
| Канцеляризми в розмові | Прості слова |
| Сленг в офіційних текстах | Нейтральна лексика |

> **Головне**
>
> - Кожен стиль має свої маркери
> - Не змішуйте стилі в одному тексті
> - Вибір стилю залежить від ситуації та адресата
> - Трансформація вимагає заміни лексики та граматики
