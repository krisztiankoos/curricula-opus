# Lesson Content

## warm-up

### Why Learn Cyrillic?

Ukrainian uses the Cyrillic alphabet - 33 letters total. Good news: many look familiar! This module covers 18 letters you can learn quickly.

The key insight: some letters LOOK like English but make DIFFERENT sounds. We call these "False Friends" - they'll trick you if you're not careful!

## presentation

### Group 1: True Friends (7 letters)

These letters look AND sound like their Latin equivalents:

| Letter | Sound | Example |
|--------|-------|---------|
| А а | A (as in "father") | мАмА |
| Е е | E (as in "bet") | мЕтро |
| І і | I (as in "machine") | таксІ |
| О о | O (as in "hot") | фОтО |
| К к | K | Кафе |
| М м | M | Мама |
| Т т | T | Тато |

Trust these! They work exactly as you'd expect.

### Group 2: False Friends (6 letters)

DANGER! These look Latin but sound completely different:

| Letter | Looks like | Actually sounds like |
|--------|------------|---------------------|
| В в | B | V (as in "van") |
| Н н | H | N (as in "no") |
| Р р | P | R (rolled, as in Spanish) |
| С с | C | S (as in "sun") |
| У у | Y | U (as in "flute") |
| Х х | X | KH (as in German "Bach") |

Memorize these! They cause the most confusion for beginners.

### Group 3: New But Easy (5 letters)

These don't exist in Latin but are straightforward:

| Letter | Sound | Memory tip |
|--------|-------|------------|
| Б б | B | Belly with a belt |
| Д д | D | Like a house |
| З з | Z | Like the number 3 |
| Л л | L | Like a tent |
| П п | P | Like a doorway/portal |
| Ф ф | F | Like a person with arms out |

## practice

### Reading Strategy

When you see a Cyrillic word:
1. Go letter by letter
2. Watch for False Friends (В=V, Н=N, Р=R, С=S, У=U, Х=KH)
3. Sound it out slowly
4. Many words are cognates - they'll sound familiar!

### Try These Words

Sound out each word. Most are international words you'll recognize:

БАНК → B-A-N-K → bank
ПАРК → P-A-R-K → park
МЕТРО → M-E-T-R-O → metro
КАФЕ → K-A-F-E → café

## production

### Reading Challenge

Read these words without looking at transliteration:

1. МАМА
2. ТАТО
3. СТУДЕНТ
4. ФУТБОЛ
5. МУЗИКА

### False Friend Drill

Remember: П = P, Р = R, В = V, Н = N, С = S, Х = KH

Read these tricky words:
- СУПЕРМАРКЕТ (supermarket)
- РЕСТОРАН (restaurant)
- ПАСПОРТ (passport)

# Activities

## match-up: False Friends Letter Sounds

> Match each Cyrillic "False Friend" letter to its actual sound. Don't be fooled by appearances!

| Left | Right |
|------|-------|
| В в | V sound (not B!) |
| Н н | N sound (not H!) |
| Р р | R sound (not P!) |
| С с | S sound (not C!) |
| У у | U sound (not Y!) |
| Х х | KH sound (not X!) |

## quiz: Read International Words

> Choose the correct English meaning for each Cyrillic word. Sound out each letter!

1. What does БАНК mean?
   - [x] Bank
   - [ ] Bunk
   - [ ] Back
   - [ ] Bonk
   > БАНК = B-A-N-K. Remember: Б=B, А=A, Н=N (not H!), К=K

2. What does ПАРК mean?
   - [x] Park
   - [ ] Pork
   - [ ] Peak
   - [ ] Pack
   > ПАРК = P-A-R-K. Remember: П=P, А=A, Р=R (not P!), К=K

3. What does МЕТРО mean?
   - [x] Metro/Subway
   - [ ] Meter
   - [ ] Metro
   - [ ] Mother
   > МЕТРО = M-E-T-R-O. All True Friends except Р=R

4. What does КАФЕ mean?
   - [x] Café
   - [ ] Coffee
   - [ ] Cave
   - [ ] Calf
   > КАФЕ = K-A-F-E. К=K, А=A, Ф=F, Е=E

5. What does ТАКСІ mean?
   - [x] Taxi
   - [ ] Task
   - [ ] Tax
   - [ ] Tuxedo
   > ТАКСІ = T-A-K-S-I. Remember: С=S (not C!), І=I

6. What does СТУДЕНТ mean?
   - [x] Student
   - [ ] Stadium
   - [ ] Study
   - [ ] Stupid
   > СТУДЕНТ = S-T-U-D-E-N-T. С=S, У=U, Н=N

7. What does ФУТБОЛ mean?
   - [x] Football
   - [ ] Footstool
   - [ ] Fútbol
   - [ ] Futon
   > ФУТБОЛ = F-U-T-B-O-L. Ф=F, У=U, Б=B, Л=L

8. What does РАДІО mean?
   - [x] Radio
   - [ ] Ready
   - [ ] Rodeo
   - [ ] Ratio
   > РАДІО = R-A-D-I-O. Remember: Р=R (not P!)
